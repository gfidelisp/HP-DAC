# utils

Presentation standards for the Hp-DAC project: how every figure and every
table in the reports, papers and deliverables is built.

The point of keeping them together is that they are one standard, not two. A
quantity carries the same symbol and the same unit whether it appears on an
axis, in a table header or in the running text. Where the two media
legitimately differ — a caption goes below a figure and above a table — the
difference is stated here rather than left to memory.

Neither standard has an API to learn. For figures you write ordinary
matplotlib; for tables you write ordinary LaTeX.
